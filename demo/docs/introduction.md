## 技术基础

Bin UI Design 基于 [bin-ui-next](https://wangbin3162.github.io/bin-ui-next/#/guide) 演进而来，可用于平滑迁移与能力升级。

组件与样式体系采用 TypeScript 与 PostCSS 重构，保持良好的可维护性与扩展性。
