# Installation

Note: bin-ui-design only supports Vue 3. If you are using Vue 2, please use [bin-ui](https://wangbin3162.github.io/bin-ui/#/guide).

## Latest Version

[![NPM version](https://img.shields.io/npm/v/bin-ui-design.svg)](https://www.npmjs.com/package/bin-ui-design)

## npm

We recommend using pnpm for installation, as it integrates better with bundlers and TypeScript. It also supports on-demand imports.

```sh
npm i bin-ui-design
# or
yarn add bin-ui-design
# or
pnpm add bin-ui-design
```

## Usage in SFC

We recommend using it in a Single File Component (SFC) environment. You can either import components directly or install them globally in your Vue app.

See [Usage in SFC](/en/docs/usage-sfc.html)
